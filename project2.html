Description

This database was developed and managed in SQL in the Third Normal Form using techniques such as stored procedures, functions, and triggers which improved my appreciation and passion in the field and cemented my knowledge in industry standards.

The database was developed over the course of a semester and gave me new knowledge on how to implement a database at an industry level.

Skills Learned

The skills that I learned during this project was entity-relationship modeling, normalization, and key management. I also gained practical proficiency in MySQL Workbench, defining constraints, selecting appropriate data types, and creating database views for reporting.
